Explicação do código

Arquivo HTML

  • <!DOCTYPE html>: Declaração do tipo de documento que informa ao navegador que este é um documento HTML5.
  • <html lang="pt-br">: Início do documento HTML, com o atributo lang especificando que o idioma do conteúdo é português do Brasil. </html>: Fim do documento HTML.
  • <head>: Contém metadados sobre o documento, como o conjunto de caracteres e a configuração de visualização. </head>: Fim da seção de metadados.
  • <meta charset="UTF-8" />: Define a codificação de caracteres como UTF-8, que suporta quase todos os caracteres e símbolos.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0" />: Configura a visualização para garantir que a página seja exibida corretamente em dispositivos móveis.
  • <title>Loading - Please Wait</title>: Define o título da página que aparece na aba do navegador.
  • <link rel="stylesheet" href="estyle.css" />: Vincula um arquivo CSS externo chamado "estyle.css" para estilizar a página.
  • <body>: Contém o conteúdo visível da página. </body>: Fim do conteúdo visível da página.
  • <div class="menu-toggle">: Cria um contêiner para o menu de navegação com a classe "menu-toggle". </div>: Fim do contêiner do menu de navegação.
  • <input type="checkbox" id="menu-check" />: Cria uma caixa de seleção oculta que será usada para alternar o menu.
  • <div class="menu-btn">: Contêiner para o botão do menu. </div>: Fim do contêiner do botão do menu.
  • <label for="menu-check">: Rótulo associado à caixa de seleção, que contém três spans para criar o ícone do menu. </label>: Fim do rótulo associado à caixa de seleção.
  • <span></span>: Três spans vazios que formam as linhas do ícone do menu.
  • <nav class="menu">: Cria um elemento de navegação com a classe "menu". </nav>: Fim do elemento de navegação.
  • <a href="#">Home</a>: Link para a página inicial.
  • <a href="#">About</a>: Link para a página "Sobre".
  • <a href="#">Services</a>: Link para a página de serviços.
  • <a href="#">Contact</a>: Link para a página de contato.

Arquivo CSS

  • body: Define o estilo global do corpo da página.
    • margin: 0;: Remove a margem padrão.
    • padding: 0;: Remove o preenchimento padrão.
  • .menu-toggle: Estiliza o contêiner do menu de navegação.
    • background-color: #333;: Define a cor de fundo como cinza escuro.
    • display: flex;: Utiliza o modelo de layout flexível.
    • flex-direction: column;: Organiza os itens em uma coluna.
    • align-items: center;: Centraliza os itens horizontalmente.
    • padding: 5px;: Adiciona um preenchimento interno de 5 pixels.
  • .menu-btn: Estiliza o botão do menu.
    • display: flex;: Utiliza o modelo de layout flexível.
    • flex-direction: column;: Organiza os itens em uma coluna.
    • justify-content: space-around;: Distribui os itens com espaço ao redor.
    • width: 30px;: Define a largura como 30 pixels.
    • height: 30px;: Define a altura como 30 pixels.
    • cursor: pointer;: Muda o cursor para um ponteiro ao passar sobre o botão.
    • position: relative;: Define a posição como relativa.
  • .menu-btn label: Estiliza o rótulo do botão do menu.
    • display: flex;: Utiliza o modelo de layout flexível.
    • flex-direction: column;: Organiza os itens em uma coluna.
    • justify-content: space-around;: Distribui os itens com espaço ao redor.
    • height: 100%;: Define a altura como 100% do contêiner pai.
  • .menu-btn span: Estiliza as linhas do ícone do menu.
    • display: block;: Exibe como um bloco.
    • width: 100%;: Define a largura como 100% do contêiner pai.
    • height: 3px;: Define a altura como 3 pixels.
    • background-color: #fff;: Define a cor de fundo como branco.
    • transition: all 0.3s ease;: Adiciona uma transição suave de 0.3 segundos.
  • #menu-check: Estiliza a caixa de seleção do menu.
    • display: none;: Oculta a caixa de seleção.
  • #menu-check:checked + .menu-btn label span:nth-child(1): Estiliza a primeira linha do ícone do menu quando a caixa de seleção está marcada.
    • transform: rotate(45deg);: Rotaciona a linha em 45 graus.
    • position: absolute;: Define a posição como absoluta.
    • top: 12px;: Define a posição superior como 12 pixels.
  • #menu-check:checked + .menu-btn label span:nth-child(2): Estiliza a segunda linha do ícone do menu quando a caixa de seleção está marcada.
    • opacity: 0;: Define a opacidade como 0, tornando a linha invisível.
  • #menu-check:checked + .menu-btn label span:nth-child(3): Estiliza a terceira linha do ícone do menu quando a caixa de seleção está marcada.
    • transform: rotate(-45deg);: Rotaciona a linha em -45 graus.
    • position: absolute;: Define a posição como absoluta.
    • top: 12px;: Define a posição superior como 12 pixels.
  • .menu: Estiliza o menu de navegação.
    • display: none;: Oculta o menu.
    • flex-direction: column;: Organiza os itens em uma coluna.
    • align-items: center;: Centraliza os itens horizontalmente.
    • margin-top: 10px;: Adiciona uma margem superior de 10 pixels.
    • padding: 5px;: Adiciona um preenchimento interno de 5 pixels.
  • #menu-check:checked ~ .menu: Estiliza o menu de navegação quando a caixa de seleção está marcada.
    • display: flex;: Exibe o menu utilizando o modelo de layout flexível.
  • .menu a: Estiliza os links do menu de navegação.
    • text-decoration: none;: Remove a decoração do texto (sublinhado).
    • color: #fff;: Define a cor do texto como branco.
    • font-size: 18px;: Define o tamanho da fonte como 18 pixels.
    • margin-bottom: 10px;: Adiciona uma margem inferior de 10 pixels.
  • .menu a:hover: Estiliza os links do menu de navegação quando o cursor está sobre eles.
    • color: #ffff00;: Define a cor do texto como amarelo.
