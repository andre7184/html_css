Explicação do código

Arquivo HTML

  • <!DOCTYPE html>Declaração do tipo de documento, especificando que é um documento HTML5.
  • <html lang="en">Início do documento HTML, com o idioma definido como inglês.
  • <head>Contém metadados sobre o documento, como o conjunto de caracteres e o título.
  • <meta charset="UTF-8">Define a codificação de caracteres como UTF-8.
  • <title>Preview</title>Define o título da página que aparece na aba do navegador.
  • <link rel="stylesheet" href="estyle.css" >caminho para o arquivo do codigo css.
  • <body>Contém o conteúdo visível da página.
  • <div class="container">Div principal que contém todos os elementos.
  • <div class="wrapper">Div que agrupa os elementos de carregamento.
  • <div class="loader"><div class="dot"></div></div>Seis divs que representam os elementos de carregamento, cada um contendo uma div interna com a classe dot.
  • <div class="text">Processando</div>Div que exibe o texto "Processando".

Arquivo CSS

  • Global Selector (*):
    • margin: 0; Remove todas as margens padrão.
    • padding: 0; Remove todos os preenchimentos padrão.
    • font-family: sans-serif; Define a fonte padrão como sans-serif.
  • body:
    • display: flex; Define o corpo como um contêiner flexível.
    • justify-content: center; Centraliza o conteúdo horizontalmente.
    • align-items: center; Centraliza o conteúdo verticalmente.
    • min-height: 100vh; Define a altura mínima como 100% da altura da janela de visualização.
    • background: #0079d7; Define a cor de fundo como azul (#0079d7).
  • .container:
    • position: relative; Define a posição como relativa.
    • width: 100%; Define a largura como 100%.
    • display: flex; Define o contêiner como flexível.
    • justify-content: center; Centraliza o conteúdo horizontalmente.
  • .container .wrapper:
    • position: absolute; Define a posição como absoluta.
    • top: -35px; Move o elemento 35 pixels para cima.
    • transform: scale(1.5); Aumenta a escala do elemento em 1.5 vezes.
  • .loader:
    • position: absolute; Define a posição como absoluta.
    • left: -25px; Move o elemento 25 pixels para a esquerda.
    • width: 50px; Define a largura como 50 pixels.
    • height: 50px; Define a altura como 50 pixels.
    • animation: rotate 3.5s linear infinite; Aplica a animação de rotação por 3.5 segundos de forma linear e infinita.
  • .loader:nth-child(n):
    • animation-delay: Xs; Define um atraso na animação para cada elemento filho.
  • @keyframes rotate:
    • 30%: Rotaciona o elemento em 220 graus.
    • 40%: Rotaciona o elemento em 450 graus.
    • 75%: Rotaciona o elemento em 720 graus e define a opacidade como 1.
    • 76%: Define a opacidade como 0.
    • 100%: Rotaciona o elemento de volta para 0 graus e define a opacidade como 0.
  • .loader .dot:
    • position: relative; Define a posição como relativa.
    • top: 30px; Move o elemento 30 pixels para baixo.
    • width: 7px; Define a largura como 7 pixels.
    • height: 7px; Define a altura como 7 pixels.
    • background: #fff; Define a cor de fundo como branco.
    • border-radius: 50%; Define bordas arredondadas, tornando o elemento circular.
  • .container .text:
    • position: absolute; Define a posição como absoluta.
    • color: #fff; Define a cor do texto como branco.
    • bottom: -85px; Move o elemento 85 pixels para baixo.
    • font-size: 25px; Define o tamanho da fonte como 25 pixels.
