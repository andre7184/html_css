Explicação do Código

HTML

<!DOCTYPE html>: Declara o tipo de documento como HTML5.

<html lang="en">: Define o idioma da página como inglês.

<head>: Contém metadados e links para estilos.

<meta charset="UTF-8">: Define a codificação de caracteres como UTF-8.

<title>Preview</title>: Define o título da página.

<style>: Contém o CSS embutido.

<body>: Contém o conteúdo visível da página.

<div class="container">: Um contêiner para centralizar o conteúdo.

<div class="wrapper">: Um contêiner para os loaders.

<div class="loader"><div class="dot"></div></div>: Cada loader contém um ponto branco.

<div class="text">Processando</div>: Texto exibido abaixo dos loaders.

CSS

margin: Define a margem de todos os elementos como 0.

padding: Define o preenchimento de todos os elementos como 0.

font-family: Define a fonte de todos os elementos como sans-serif.

body

display: Define o tipo de layout como flexível (flex).

justify-content: Centraliza o conteúdo horizontalmente.

align-items: Centraliza o conteúdo verticalmente.

min-height: Define a altura mínima como 100% da altura da janela de visualização (viewport).

background: Define a cor de fundo como azul (#0079d7).

.container

position: Define a posição como relativa.

width: Define a largura como 100%.

display: Define o tipo de layout como flexível (flex).

justify-content: Centraliza o conteúdo horizontalmente.

.wrapper

position: Define a posição como absoluta.

top: Define a posição superior como -35px.

transform: Aplica uma escala de 1.5 ao elemento.

.loader

position: Define a posição como absoluta.

left: Define a posição à esquerda como -25px.

width: Define a largura como 50px.

height: Define a altura como 50px.

animation: Aplica a animação “rotate” com duração de 3.5 segundos, linear e infinita.

.loader:nth-child(n)

animation-delay: Define um atraso na animação para cada loader, variando de 0.15s a 0.90s.

@keyframes rotate

30%: Rotaciona o elemento em 220 graus.

40%: Rotaciona o elemento em 450 graus.

75%: Rotaciona o elemento em 720 graus e define a opacidade como 1.

76%: Define a opacidade como 0.

100%: Retorna a rotação para 0 graus e a opacidade para 0.

.loader .dot

position: Define a posição como relativa.

top: Define a posição superior como 30px.

width: Define a largura como 7px.

height: Define a altura como 7px.

background: Define a cor de fundo como branco (#fff).

border-radius: Define os cantos como arredondados (50%).

.container .text

position: Define a posição como absoluta.

color: Define a cor do texto como branco (#fff).

bottom: Define a posição inferior como -85px.

font-size: Define o tamanho da fonte como 25px.

Esses atributos juntos criam uma animação de carregamento com pontos giratórios e um texto “Processando” centralizado na tela.
