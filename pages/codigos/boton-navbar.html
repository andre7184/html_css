Codigo html/css - Botao Navbar

Explicação do código

Arquivo HTML

  • <!DOCTYPE html>: Declara o tipo de documento como HTML5.
  • <html>: Inicia o documento HTML.
  • <head>: Contém metadados e links para recursos externos.
  • <meta charset="UTF-8">: Define a codificação de caracteres como UTF-8.
  • <meta http-equiv="X-UA-Compatible" content="IE=edge">: Garante a compatibilidade com o modo de renderização mais recente do Internet Explorer.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Configura a viewport para garantir que a página seja responsiva em dispositivos móveis.
  • <title>Bottom Navigation Bar</title>: Define o título da página que aparece na aba do navegador.
  • <link rel="stylesheet" href="estyle.css">: Link para um arquivo CSS externo chamado "estyle.css".
  • <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.2.1/css/all.min.css" />: Link para a biblioteca de ícones Font Awesome.
  • <body>: Inicia o corpo do documento HTML.
  • <div class="menu">: Cria um contêiner div com a classe "menu".
  • <ul>: Inicia uma lista não ordenada.
  • <li>: Cria um item de lista.
  • <a href="">: Cria um link (ainda sem URL definida).
  • <span class="icon">: Cria um contêiner span com a classe "icon" para ícones.
  • <i class="fa-solid fa-house-user"></i>: Usa um ícone da biblioteca Font Awesome (ícone de casa).
  • <span class="title">Home</span>: Cria um contêiner span com a classe "title" para o texto "Home".
  • Repetição dos itens de lista: Os itens de lista são repetidos para "Wallet", "Scan", "Cart" e "Account", cada um com ícones e textos correspondentes.
  • Fechamento das tags: As tags são fechadas na ordem inversa de abertura: </ul>, </div>, </body>, </html>.

Arquivo CSS

  • body:
    • background-color: #9ae7bd; Define a cor de fundo como um tom de verde claro.
    • height: 100vh; Define a altura como 100% da altura da janela de visualização.
    • display: flex; Define o corpo como um contêiner flexível.
    • justify-content: center; Centraliza o conteúdo horizontalmente.
    • align-items: center; Centraliza o conteúdo verticalmente.
  • .menu ul:
    • display: flex; Define a lista como um contêiner flexível.
    • list-style: none; Remove os marcadores padrão da lista.
    • margin: 0; Remove as margens padrão.
    • padding: 0 30px; Define o preenchimento superior e inferior como 0 e o preenchimento lateral como 30 pixels.
    • background-color: #006105; Define a cor de fundo como verde escuro.
    • border-radius: 12px; Arredonda os cantos da lista com um raio de 12 pixels.
    • box-shadow: 0 0 10px rgba(0, 0, 0, 0.5); Adiciona uma sombra ao redor da lista com um desfoque de 10 pixels e uma opacidade de 50%.
  • .menu ul li a:
    • color: #fff; Define a cor do texto como branco.
    • display: flex; Define o link como um contêiner flexível.
    • flex-direction: column; Organiza os itens dentro do link em uma coluna.
    • font-size: 20px; Define o tamanho da fonte como 20 pixels.
    • font-family: poppins; Define a fonte como "Poppins".
    • text-decoration: none; Remove a decoração padrão do texto (sublinhado).
    • height: 70px; Define a altura do link como 70 pixels.
    • width: 70px; Define a largura do link como 70 pixels.
    • position: relative; Define a posição como relativa.
  • .menu ul li a .title:
    • position: absolute; Define a posição como absoluta.
    • font-size: 16px; Define o tamanho da fonte como 16 pixels.
    • top: 50%; Posiciona o elemento 50% a partir do topo do contêiner pai.
    • left: 50%; Posiciona o elemento 50% a partir da esquerda do contêiner pai.
    • transform: translate(-50%, -50%); Centraliza o elemento no contêiner pai.
    • opacity: 0; Define a opacidade como 0 (invisível).
    • transition: .5s; Define uma transição suave de 0.5 segundos.
  • .menu ul li a .icon:
    • position: absolute; Define a posição como absoluta.
    • font-size: 20px; Define o tamanho da fonte como 20 pixels.
    • top: 50%; Posiciona o elemento 50% a partir do topo do contêiner pai.
    • left: 50%; Posiciona o elemento 50% a partir da esquerda do contêiner pai.
    • transform: translate(-50%, -50%); Centraliza o elemento no contêiner pai.
    • transition: .5s; Define uma transição suave de 0.5 segundos.
    • width: 50px; Define a largura do ícone como 50 pixels.
    • height: 50px; Define a altura do ícone como 50 pixels.
    • border-radius: 50%; Arredonda os cantos do ícone, tornando-o circular.
    • display: flex; Define o ícone como um contêiner flexível.
    • justify-content: center; Centraliza o conteúdo do ícone horizontalmente.
    • align-items: center; Centraliza o conteúdo do ícone verticalmente.
  • .menu ul li a:hover .title:
    • transform: translate(-50%, calc(-50% + 10px)); Move o título 10 pixels para baixo.
    • opacity: 1; Define a opacidade como 1 (visível).
  • .menu ul li a:hover .icon:
    • top: 0; Move o ícone para o topo do contêiner pai.
    • background-color: #55eb5d; Define a cor de fundo do ícone como verde claro.
    • color: #fff; Define a cor do ícone como branco.
    • outline: 4px #fff solid; Adiciona uma borda branca sólida de 4 pixels ao redor do ícone.
